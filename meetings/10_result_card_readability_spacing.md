# Result Card Readability And Spacing

Date: 2026-06-05

## Changes

- Increased recommendation card typography again for names, tags, score details, recipe headings, and recipe body text.
- Reduced the recommendation panel padding and result grid gaps so cards sit closer together and closer to the result frame edges.
- Changed score dimensions to a label/value row structure so each dimension title and score remain on the same line.
- Added controlled spacing for two-card and three-card result rows while preserving fixed card widths.
